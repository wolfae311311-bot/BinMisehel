# بن مسحل - Satellite Master Chat 🛰️

تطبيق دردشة فوري متقدم باللغة العربية مع واجهة حديثة وجميلة

## ✨ المميزات

- 💬 **دردشة فورية**: تواصل حي مع المستخدمين الآخرين
- 🏰 **عدة غرف**: 6 غرف دردشة مختلفة (المجلس العام، VIP، مختبر التقنية، سوالف الورد، الإدارة، شقة بن مسحل)
- 👥 **قائمة الأعضاء**: اعرض جميع الأعضاء المتصلين حالياً
- 🎨 **واجهة عربية**: تصميم حديث وسهل الاستخدام
- ☔ **تأثيرات بصرية**: تأثير المطر الديناميكي
- 🔔 **إخطارات الاتصال**: معرفة حالة الاتصال بالخادم
- 📱 **متوافق مع الأجهزة**: يعمل على الهاتف والويب

## 📝 خطوات البدء

1. **افتح التطبيق** من أحد الروابط أعلاه
2. **اكتب اسمك** في حقل "اكتب اسمك للمباشرة"
3. **اضغط تفضل** أو اضغط Enter
4. **اختر غرفة** من الغرف المتاحة
5. **ابدأ الدردشة**! 💬

## 🏠 الغرف المتاحة

| الغرفة | الرمز | الوصف |
|-------|------|-------|
| 🏠 المجلس العام | r1 | غرفة عامة للجميع |
| ⭐ جناح VIP | r2 | غرفة VIP الحصرية |
| 🧪 مختبر التقنية | r3 | نقاشات تقنية |
| 🌿 سوالف الورد | r4 | حوارات عامة |
